# Changelog

## v0.5.0

### 新增

- 新增连续捕捉模式，可在用户连续点击多个页面、弹窗、浮层后生成一份汇总报告。
- 新增连续捕捉进度与状态反馈，支持开始、停止、取消与汇总完成提示。
- 新增提示词模板中心，内置 `通用` 与 `设计体验` 两套模板。
- 新增提示词模板持久化存储，支持保存当前模板、另存模板、删除模板、恢复初始。
- 新增报告截图与连续捕捉步骤截图预览，支持在侧边栏内回看。

### 体验优化

- 统一 `options`、`popup`、`sidepanel` 三处配置界面的交互和视觉风格，整体调整为更接近 shadcn 的中性卡片样式。
- 配置卡片改为默认折叠，可点击标题展开，减少默认信息噪音。
- “将检查的问题在页面高亮显示”改为顶层开关，并优化与页面高亮逻辑的联动。
- “额外规则”调整为单独卡片，放到配置表单底部。
- “模型超时毫秒”移动到模型配置卡片底部，配置结构更清晰。
- 提示词模板区改为 `单页` / `连续捕捉` Tab 切换，减少长表单滚动。
- 模板选择后立即切换内容，编辑时保持当前模板选中状态，不再自动回退到通用模板。

### 兼容与实现

- 新增 `prompt-templates.js`，统一管理默认提示词、内置模板与模板存储格式。
- 扩展权限中新增 `alarms`，用于连续捕捉过程中的状态控制与超时管理。
- 更新 Apple Notes bridge、后台脚本与内容脚本，补充连续捕捉、截图和导出相关能力。
